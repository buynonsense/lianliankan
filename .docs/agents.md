# 项目介绍

这是一个基于 Next.js 15 构建的连连看游戏项目，采用“柔和唯美主义 (Soft Aestheticism)”设计风格，强调玻璃态视觉、微光特效和丝滑的交互体验。

# 架构

- **前端**: Next.js 15 (App Router), React 19, Tailwind CSS 4
- **动画**: Framer Motion
- **图标**: Lucide React
- **持久化**: Prisma, Postgres

# 设计契约 (Design Contract)

- **调色板**: 桃粉色 (#f8ad9d)、暖桔色 (#fbc4ab)、薰衣草紫 (#fbc4ab 等渐变)
- **边框**: 统一使用 `rounded-[2.5rem]` 或 `rounded-3xl` 的大圆角
- **视觉反馈**: 悬浮发光、点击缩放、全屏渐入转场
- **核心组件**:
  - `.glass`: 玻璃拟态效果（模糊背景 + 半透明白色边框）
  - `.btn-primary`: 带有呼吸感的品牌色按钮
